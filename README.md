# 《凤谋：穿越王妃带球跑》| Phoenix Strategy: The Transmigrated Princess

> 一场跨越千年的爱恋，一段先婚后爱的虐恋深情。
> 
> 她带着球跑了，他追妻火葬场。

---

## 📖 小说简介 | Synopsis

《凤谋：穿越王妃带球跑》是一部古代穿越言情小说，讲述现代女孩苏念意外穿越到古代，与冷面王爷萧煜之间跌宕起伏的爱情故事。

**核心设定**：先孕后爱 + 虐恋情深 + 破镜重圆

---

## 👥 主要人物 | Main Characters

### 女主：苏念 (Su Nian)
- **身份**：现代穿越者，21世纪普通白领
- **性格**：聪明独立、坚强隐忍、内心柔软
- **特长**：懂医术、会做饭、有现代人的思维方式
- **命运**：意外穿越后怀孕，陷入困境

### 男主：萧煜 / 煜王 (Xiao Yu / Prince Yu)
- **身份**：当朝王爷，大周朝战神
- **性格**：外冷内热、霸道深情、不善表达
- **命运**：被迫卷入政治漩涡，与女主经历误会分离

### 男二：顾言 / 耶律齐 (Gu Yan / Yelü Qi)
- **身份**：北狄国王子，伪装成太医院院判之子
- **特点**：温润如玉的外表下隐藏着深沉的心机
- **作用**：推动剧情发展，制造冲突

### 女配：柳如烟 (Liu Ru Yan)
- **身份**：王府侧妃
- **性格**：心机深沉，步步为营
- **行为**：处处针对女主，散布谣言
